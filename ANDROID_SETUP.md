# Expense AI Android Setup

1. Run `npm install` in the project root.
2. Run `npx cap add android`.
3. Run `npx cap sync android`.
4. Run `npx cap open android`.
5. Keep the backend running with `node BACKEND\server.js`.
6. In the app, use `http://10.0.2.2:3000` for the Android emulator.
7. On a real phone, use your laptop IP like `http://192.168.1.5:3000`.
